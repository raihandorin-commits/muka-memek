# Muka Memek - Galeri Foto dan Musik

Halaman web interaktif untuk menampilkan galeri foto dan musik.

## 📋 Fitur

- 🎵 **Musik Player** - Putar musik favorit Anda
- 📷 **Galeri Foto** - Tampilkan foto-foto dengan efek hover
- 📱 **Responsive Design** - Tampil sempurna di semua perangkat
- ✨ **Modern UI** - Desain modern dengan gradient dan animasi

## 🚀 Cara Menggunakan

### 1. Aktifkan GitHub Pages
- Buka Settings → Pages
- Pilih Branch: `main`
- Pilih Folder: `/ (root)`
- Simpan

Web Anda akan tersedia di: `https://raihandorin-commits.github.io/muka-memek/`

### 2. Tambahkan Foto dan Musik Anda

Buat folder `assets` dan masukkan file:

```
assets/
├── foto1.jpg
├── foto2.jpg
├── foto3.jpg
├── foto4.jpg
└── music.mp3
```

### 3. Edit Konten (Opsional)

- Buka `index.html` untuk mengedit judul dan deskripsi
- Buka `style.css` untuk mengubah warna dan desain
- Tambahkan lebih banyak foto dengan menyalin blok `.gallery-item`

## 📁 Struktur File

```
muka-memek/
├── index.html        # Halaman utama
├── style.css         # Styling
├── script.js         # JavaScript
├── README.md         # File ini
└── assets/
    ├── foto1.jpg
    ├── foto2.jpg
    ├── foto3.jpg
    ├── foto4.jpg
    └── music.mp3
```

## 🎨 Kustomisasi Warna

Ubah kode warna di `style.css`:

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
/* Ganti dengan warna yang Anda inginkan */
```

## 📝 Catatan

- Pastikan file foto dalam format JPG/PNG
- File musik dalam format MP3/WAV
- Ukuran file optimal:
  - Foto: Max 2MB per file
  - Musik: Max 10MB per file
